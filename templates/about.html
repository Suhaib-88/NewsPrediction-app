About this project

Duration of the project

This project was started on 23/09/21 and completed on 08/10/21

Inspiration of this project

When i started looking for different ideas for my first project, I came across this dataset on kaggle called "News Category Dataset". Then I bought into the idea, designed the architechture and workflow of the complete project.

What is the purpose of this app ?

It's main purpose is to help people understand the context of the news article. Why? Because more often people recieve a piece of information on whatapp and have no idea. So if we can understand where the piece of information is coming from, it becomes easier to digest the information.Also sometimes a news article may belong to one or more categories, so knowing which category it mainly comes from is crucial. This APP also helps in labelling the news with tags. Tags include: person,country/nationality, organisation name or simply date of an event. It helps readers in understanding context furthermore.

Technology behind this app?

This app uses machine learning technology to help learn relationship between news headlines and news category. The machine learning technology used in this project is called Multinomial Naives Bayes which works great with text data.

Why use machine learning model & not NLP model?

The main reason i choose machine learning model is due to its lesser complexity, training time and faster adaptability. Deep learning models requires training data on spot and then predicting. While machine learning models can be pretrained and predictions can be done right away.

Challenges faced during the project

The most challenging and time-taking part of this project was deciding which technology to use. Should i use Natural Language processing? or Artificial neural networks? or Machine Learning Model?

Mistakes made during this project

My biggest mistake made was wrongly selecting Natural Language processing model and going with a much complex model straight away, while much easier options were available. It cost me a lot of time and finally i realised that a much simpler solution could do the work.
